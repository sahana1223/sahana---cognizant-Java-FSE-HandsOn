# Exercise 6 - Stored Procedures

## Scenario 1
Create a stored procedure to process monthly interest for savings accounts.

## Scenario 2
Create a stored procedure to update employee salary based on department and bonus percentage.

## Scenario 3
Create a stored procedure to transfer funds between accounts after checking sufficient balance.

## Concepts Used

- CREATE PROCEDURE
- Parameters
- UPDATE
- IF Statement
- COMMIT
- DBMS_OUTPUT
